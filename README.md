This repository contains R code and data as well as Gephi´s (https://gephi.org/) files and output networks, serving as a basis for all the analysis in the manuscript "From colonial legacy to global integration: How war transformed Ukrainian vertebrate palaeontology"
by Viktor Baranov1,  Oleksandr Kovalchuk, Leonid Gorobets, Paul P. Stewens, Svitozar Davydenko, Pavlo Otriazhyi, Emma M. Dunne, Tymofii Sokolskyi, Laetitia Navarro and Pavel Gol’din
All code by V. Baranov
Repository contains following parts
"geodiversity_clean"

"geodiversity_clean\gephi networks" #data for the network analysis in Gephi

-"geodiversity_clean\gephi networks\Network files" # network files plotted from the analysis
"edges.xlsx" #network edges for the general cooperation network of publications in Ukrainian vertebrate palaeontology
"edges_dynamic_gephi.csv" #network edges for the dynamic (decade -by-decade) cooperation network of publications in Ukrainian vertebrate palaeontology
"nodes.xlsx" #network nodes for the general cooperation network of publications in Ukrainian vertebrate palaeontology
"nodes_dynamic.csv"  #network nodes  for the dynamic (decade -by-decade) cooperation network of publications in Ukrainian vertebrate palaeontology

-"geodiversity_clean\gephi networks\Base data"
"general network.gephi" #plot for the general cooperation network of publications in Ukrainian vertebrate palaeontology
"general network_decadal.gephi" #plot for the dynamic (decade -by-decade) cooperation network of publications in Ukrainian vertebrate palaeontology
"ua post 2014 network.gephi"  #plot for the post 2014 cooperation network of publications in Ukrainian vertebrate palaeontology
"crimea network.gephi" #plot for the post 2014 cooperation network of publications in Ukrainian vertebrate palaeontology (based on fossils from Crimea)

-"geodiversity_clean\R Data"
"main bibliographic database.xlsx" raw bibliographic database
"crimea_4.csv" #bibliographic data and dataset for publications on Crimean vertebrate fossils post 2014
"df2.csv" # general bibliographic data and dataset for publications on the rest of Ukrainian vertebrate fossils
"edges_crimea.csv"# edges for the post 2014 cooperation network of publications in Ukrainian vertebrate palaeontology (based on fossils from Crimea)
"edges_ua14.csv" # edges for the post 2014 cooperation network of publications in Ukrainian vertebrate palaeontology (based on fossils from the rest of Ukraine)
"nodes_ua14.csv"#  for nodes the post 2014 cooperation network of publications in Ukrainian vertebrate palaeontology (based on fossils from the rest of Ukraine)
"nodes1_crimea.csv"# nodes for the post 2014 cooperation network of publications in Ukrainian vertebrate palaeontology (based on fossils from Crimea)
"ua_14.csv" #bibliographic data and dataset for publications for the rest of Ukrainian vertebrate fossils post 2014
